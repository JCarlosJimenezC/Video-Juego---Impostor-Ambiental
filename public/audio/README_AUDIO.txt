INSTRUCCIONES PARA ARCHIVOS DE AUDIO
=====================================

Esta carpeta debe contener los siguientes archivos de audio .mp3:

- click.mp3       : Sonido suave al hacer clic en botones
- acierto.mp3     : Fanfarria de victoria / celebración
- error.mp3       : Sonido de fallo / incorrecto
- votacion.mp3    : Sonido de transición a votación (tambores / suspenso)

FUENTES GRATUITAS RECOMENDADAS (licencia libre):
- Freesound.org (Creative Commons)
- Pixabay.com/sound-effects (libre de derechos)
- Mixkit.co (uso gratuito)

El juego funciona sin estos archivos (los errores de audio se capturan silenciosamente),
pero se recomienda agregarlos para cumplir el requisito multimedia del IF7102.
