## 4. Uso de Inteligencia Artificial

Durante el desarrollo de este proyecto se usó **Claude (Anthropic)** como asistente de programación para:

- Generar la estructura inicial de componentes Vue 3
- Explicar conceptos del Composition API (`ref`, `computed`, ciclo de vida)
- Revisar y corregir errores de sintaxis en templates
- Generar el contenido del archivo `juego-data.json` con palabras ambientales y sus pistas
- Diseñar la lógica de puntuación del juego

Todo el código generado fue revisado, comprendido y adaptado por el estudiante. Se siguieron las instrucciones del docente respecto al uso de IA como apoyo (no como sustituto del aprendizaje).
